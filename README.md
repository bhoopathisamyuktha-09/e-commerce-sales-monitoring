# E-Commerce Sales Monitoring Project

## Project Overview
This project focuses on monitoring and analyzing e-commerce sales data using Excel and Power BI. It was developed as part of the Artificial Intelligence Development Associate course.

## Objectives
- Analyze sales performance
- Identify revenue trends
- Monitor product and category-wise sales
- Support business decision-making using visual dashboards

## Tools Used
- Microsoft Excel
- Microsoft Power BI

## Dataset
- File Name: E-Commerce sales monitoring.csv
- The dataset contains sales-related information such as products, categories, revenue, and time-based data.

## Key Features
- Sales trend analysis
- Category-wise performance
- Visual dashboards created in Power BI
- Cleaned and structured data using Excel
